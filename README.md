# ASCII Art Generator

This Python application converts images into artistic ASCII text. Users can customize resolution, brightness mapping, and save the generated ASCII art as a `.txt` file. Built with an intuitive Tkinter-based GUI, the application is simple and fun to use.

## Features
- Convert images to ASCII art using customizable brightness-to-character mapping.
- Adjust resolution to control ASCII art density and detail.
- Real-time preview of the generated ASCII art.
- Save the ASCII art as a `.txt` file.

## Requirements
- Python 3.x
- Required Python library: `Pillow`
